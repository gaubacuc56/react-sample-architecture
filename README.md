## Installation

### Pre-requisites

1. Node.js v21.xx.
2. Download from [here](https://nodejs.org/en/download/) or use nvm.

### How to run

1. Clone the repo:

```bash
git clone https://github.com/gaubacuc56/react-sample-architecture.git
git checkout master
```

2. Install project's dependencies and libraries:

```bash
npm install
```

3. Start application:

```bash
npm start
```
